商店页面启用伪静态URL： <{setting key="system.seo.emuStatic" namespace="setting" display='false'}>     <{help}>伪静态指的是动态网页通过重写URL的方法实现去掉动态网页的参数，但在实际的网页目录中并不存在重写网址的页面文件。

伪静态主要目的是提高搜索引擎蜘蛛(Spider)对网站网页抓取的效果，达到搜索引擎优化的目的。

注意：开启前请与主机提供商确认服务器是否支持网址重写(Rewrite)<{/help}>
